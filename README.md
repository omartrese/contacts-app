#CONTACTS APP

This is a contacts app made with node.js and express, using mySQL as RDBMS and Handlebars as 
HTML server-side rendering engine 

# DEVELOPMENT SET-UP

## 1. Clone the project
`cd contacts-app`

`npm init`

## 2. Create the database and its tables
 ***Execute the `/database/db.sql` script***

## 3. Run the app

`npm run run`
